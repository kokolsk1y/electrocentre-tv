## Цель проекта
Цифровой эфир для Smart TV в магазине электроники.
Телевизоры показывают слайды с товарами, акциями, видео и фото из Telegram-канала.
В будущем — добавить радио (фоновая музыка + промо-объявления).

---

## Stack
- **Интерфейс:** HTML-файл (одностраничное приложение) — актуальная версия `electrocentre-tv-v13.html`
- **Источник контента:** Telegram-канал (посты с фото/видео)
- **Получение медиа:** n8n Cloud + Telegram Bot API
- **Хостинг:** GitHub Pages — kokolsk1y.github.io/electrocentre-tv/
- **TTS (промо):** Яндекс SpeechKit (планируется)
