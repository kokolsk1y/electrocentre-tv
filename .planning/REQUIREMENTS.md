# Requirements — ЭлектроЦентр Smart TV Display

## V1 — Must Ship (до конца марта 2026)

| ID | Требование | Примечания |
|----|-----------|-------|
| SLIDE-01 | 9 слайдов в цикле: 4 HTML-информационных + 5 баннеров-акций | Порядок: HTML / баннер / HTML / баннер / HTML / баннер / HTML / баннер / баннер |
| SLIDE-02 | Каждый слайд показывается 10 секунд | Баннеры — 13 сек (время на восприятие фото) |
| SLIDE-03 | Плавный переход между слайдами (cross-fade 400мс) | Заменить текущую белую вспышку |
| SLIDE-04 | Анимация элементов на HTML-слайдах появляется по очереди | Уже реализовано, не ломать |
| CONTENT-01 | Слайд 1: Заказывай онлайн / STV39.RU + QR-код | Тёмно-синий фон, крупный текст |
| CONTENT-02 | Слайд 3: Скидка при разовой покупке на сайте | Таблица скидок 3–15% |
| CONTENT-03 | Слайд 5: Как получить карту постоянного клиента | 3 уровня: 6%, 8%, 10% |
| CONTENT-04 | Слайд 7: Бесплатно доставим — два варианта + условия | Заказ на сайте от 20 000 ₽, из магазина от 50 000 ₽, условия крупным текстом |
| CONTENT-05 | Слайды 2,4,6,8,9: баннеры-акции как полноэкранные изображения | Текущие 5 баннеров из папки slides/ |
| TECH-01 | Работает на Android TV без установки плагинов | Открывается в браузере TV по URL |
| TECH-02 | Разрешение 1920×1080, корректное масштабирование | transform-scale при нестандартном viewport |
| TECH-03 | Шапка: логотип, телефон, сайт, часы | Всегда видна поверх слайдов |
| TECH-04 | Бегущая строка внизу с ключевой информацией | Режим работы, доставка, юр. лица и т.д. |
| TECH-05 | Файл работает при открытии с GitHub Pages | kokolsk1y.github.io/electrocentre-tv/ |
| APPROVE-01 | Одобрено руководством на собрании | Маркетолог + директор + другие |

## V2 — Планируется после одобрения v1

| ID | Требование | Примечания |
|----|-----------|-------|
| MGMT-01 | Самостоятельное обновление баннеров без разработчика | Менеджер меняет файлы, TV подхватывает автоматически |
| MGMT-02 | Смена баннеров занимает не более 15 минут для нетехнического сотрудника | Простой процесс: загрузить файл, обновилось |
| SLIDE-05 | Скринсейвер в нерабочие часы | Защита от burn-in, движущийся логотип |
| SLIDE-06 | Дата в шапке рядом с часами | Живой вид экрана |
| CONTENT-06 | Навигационный ролик по магазину как отдельный слайд | После съёмки |

## Out of Scope (v1)

- Аудио любого вида (музыка, TTS, джинглы) — отложено до решения по лицензии РАО/ВОИС
- Аналитика просмотров
- Управление слайдами с телефона
- Таймер акции на баннерах
- TG-интеграция
- Погода
- Отзывы покупателей
